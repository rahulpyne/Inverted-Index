     <p>
      <b>
       Energy Efficiency Services Limited (EESL)
      </b>
      is the organisation responsible for implementation of
      <a href="/wiki/Domestic_Efficient_Lighting_Programme" title="Domestic Efficient Lighting Programme">
       Domestic Efficient Lighting Programme
      </a>
      (DELP) or the UJALA scheme, as it is now being referred to.
      <sup class="reference" id="cite_ref-1">
       <a href="#cite_note-1">
        [1]
       </a>
      </sup>
     </p>
     <h2>
      <span class="mw-headline" id="About">
       About
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Energy_Efficiency_Services_Limited&amp;action=edit&amp;section=1" title="Edit section: About">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      <b>
       Energy Efficiency Services Limited (EESL)
      </b>
      is a
      <a href="/wiki/Joint_venture" title="Joint venture">
       joint venture
      </a>
      of state-run power companies, responsible for implementation of
      <a href="/wiki/Domestic_Efficient_Lighting_Programme" title="Domestic Efficient Lighting Programme">
       Domestic Efficient Lighting Programme
      </a>
      (DELP).
      <sup class="reference" id="cite_ref-2">
       <a href="#cite_note-2">
        [2]
       </a>
      </sup>
      It has reduced the prices of
      <a class="mw-redirect" href="/wiki/LED_bulbs" title="LED bulbs">
       LED bulbs
      </a>
      by 75%.
      <sup class="reference" id="cite_ref-3">
       <a href="#cite_note-3">
        [3]
       </a>
      </sup>
     </p>
