   <h1 class="firstHeading" id="firstHeading" lang="en">
    Organizing principle
   </h1>
   <div class="mw-body-content" id="bodyContent">
    <div id="siteSub">
     From Wikipedia, the free encyclopedia
    </div>
    <div id="contentSub">
    </div>
    <div class="mw-jump" id="jump-to-nav">

    <div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en">
     <p>
      An
      <b>
       organizing principle
      </b>
      is a core assumption from which everything else by proximity can derive a classification or a value. It is like a central reference point that allows all other objects to be located, often used in a
      <a href="/wiki/Conceptual_framework" title="Conceptual framework">
       conceptual framework
      </a>
      . Having an organizing principle might help one simplify and get a handle on a particularly complicated domain or phenomenon. On the other hand, it might create a deceptive prism that colors one's judgment.
     </p>
     <h2>
      <span class="mw-headline" id="Examples">
       Examples
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Organizing_principle&amp;action=edit&amp;section=1" title="Edit section: Examples">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <ul>
      <li>
       In a
       <a class="mw-redirect" href="/wiki/The_Brookings_Institution" title="The Brookings Institution">
        Brookings Institution
       </a>
       article,
       <a href="/wiki/James_Steinberg" title="James Steinberg">
        James Steinberg
       </a>
       describes how
       <a class="mw-redirect" href="/wiki/Counter_terrorism" title="Counter terrorism">
        Counter terrorism
       </a>
       has become the organizing principle of U.S. national security.
       <sup class="reference" id="cite_ref-1">
        <a href="#cite_note-1">
         [1]
        </a>
       </sup>
      </li>
      <li>
       The idea of the
       <a class="mw-redirect" href="/wiki/Solar_system" title="Solar system">
        solar system
       </a>
       is based on the organizing principle that the sun is located at a central point, and all planets rotate around it.
      </li>
      <li>
       Most modern
       <a class="mw-redirect" href="/wiki/Cities" title="Cities">
        cities
       </a>
       are based on the organizing principle of the
       <a href="/wiki/Grid_plan" title="Grid plan">
        Grid plan
       </a>
       in order to better manage transportation and
       <a href="/wiki/Address_(geography)" title="Address (geography)">
        addressing
       </a>
       .
      </li>
      <li>
       Most religions (as opposed to
       <a href="/wiki/Cult" title="Cult">
        cults
       </a>
       ) can be described by social scientists as built around an organizing principle (for example, the divinity of Christ) that allows for the sustainable or improvable
       <a href="/wiki/Recursion" title="Recursion">
        recursion
       </a>
       of a unique population.
       <sup class="reference" id="cite_ref-2">
        <a href="#cite_note-2">
         [2]
        </a>
       </sup>
      </li>
      <li>
       Organizations can be constructed around a set of organizing principles, such as concepts, priorities, or goals. For example, an organization may intend to be innovative, international, quality and agile.
      </li>
      <li>
       The central organising principle of the
       <a href="/wiki/Welsh_Government" title="Welsh Government">
        Welsh Government
       </a>
       is
       <a href="/wiki/Sustainability" title="Sustainability">
        Sustainability
       </a>
       .
      </li>
      <li>
       <a href="/wiki/Sociology_of_knowledge#Legitimation_Code_Theory" title="Sociology of knowledge">
        Legitimation Code Theory
       </a>
       is an explanatory framework in the
       <a href="/wiki/Sociology_of_knowledge" title="Sociology of knowledge">
        sociology of knowledge
       </a>
       and education that seeks to understand different social fields of practices in terms of their operating principles, which determines the basis of success and failure.
       <sup class="reference" id="cite_ref-3">
        <a href="#cite_note-3">
         [3]
        </a>
       </sup>
      </li>
     </ul>
