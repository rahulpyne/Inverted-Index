    Lower Snake River Wind Project
   </h1>

    <div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en">
     <p>
      The
      <b>
       Lower Snake River Wind Project
      </b>
      is under construction in Washingtonâ€™s
      <a href="/wiki/Garfield_County,_Washington" title="Garfield County, Washington">
       Garfield
      </a>
      and
      <a href="/wiki/Columbia_County,_Washington" title="Columbia County, Washington">
       Columbia counties
      </a>
      . The
      <a href="/wiki/Wind_farm" title="Wind farm">
       wind farm
      </a>
      is being developed in phases with the first phase consisting of 149 Siemens
      <a href="/wiki/Wind_turbine" title="Wind turbine">
       wind turbines
      </a>
      rated at 2.3 MW each for a maximum generating capacity of 343 MW.
      <sup class="reference" id="cite_ref-PSElowersnake_1-0">
       <a href="#cite_note-PSElowersnake-1">
        [1]
       </a>
      </sup>
      <sup class="reference" id="cite_ref-lowers_2-0">
       <a href="#cite_note-lowers-2">
        [2]
       </a>
      </sup>
      The project will also create up to 150 temporary jobs and the installation will create up to 25 permanent jobs.
      <sup class="reference" id="cite_ref-lowers_2-1">
       <a href="#cite_note-lowers-2">
        [2]
       </a>
      </sup>
      <sup class="reference" id="cite_ref-PSEpdf_3-0">
       <a href="#cite_note-PSEpdf-3">
        [3]
       </a>
      </sup>
     </p>
