     <p>
      The
      <b>
       International Network for Sustainable Energy
      </b>
      (
      <b>
       INFORSE
      </b>
      ) is an
      <a href="/wiki/International" title="International">
       international
      </a>
      ,
      <a class="mw-redirect" href="/wiki/Non-profit_organisation" title="Non-profit organisation">
       non-profit organisation
      </a>
      (NGO) network of
      <a href="/wiki/Environmental_organization" title="Environmental organization">
       environmental organizations
      </a>
      promoting
      <a href="/wiki/Sustainable_energy" title="Sustainable energy">
       sustainable energy
      </a>
      to protect
      <a href="/wiki/Environment_(biophysical)" title="Environment (biophysical)">
       the environment
      </a>
      and to decrease
      <a href="/wiki/Poverty" title="Poverty">
       poverty
      </a>
      .
      <sup class="reference" id="cite_ref-1">
       <a href="#cite_note-1">
        [1]
       </a>
      </sup>
     </p>

     <h2>
      <span class="mw-headline" id="Activities">
       Activities
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [

       <a href="/wiki/Efficient_energy_use" title="Efficient energy use">

        <a href="#cite_note-2">
         [2]
        </a>

        <a href="#cite_note-3">

        <a href="#cite_note-4">

        <a href="#cite_note-5">

     <h2>
      <span class="mw-headline" id="History">
       History
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=International_Network_for_Sustainable_Energy&amp;action=edit&amp;section=2" title="Edit section: History">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      INFORSE was established in 1992 at the Global Forum, which was a parallel forum to the
      <a class="mw-redirect" href="/wiki/UNCED" title="UNCED">
       UNCED
      </a>
      Conference so called
      <a href="/wiki/Earth_Summit" title="Earth Summit">
       Earth Summit
      </a>
      . In 1998, INFORSE got consultative status to United Nation's
      <a class="mw-redirect" href="/wiki/ECOSOC" title="ECOSOC">
       ECOSOC
      </a>
      .
      <sup class="reference" id="cite_ref-16">
       <a href="#cite_note-16">
        [16]
       </a>
      </sup>
      INFORSE is working through Regional Coordinators in Asia, Africa, Europe, Americas. The INFORSE Secretariat is in Denmark. In 2002,
      <a href="/wiki/INFORSE-Europe" title="INFORSE-Europe">
       INFORSE-Europe
      </a>
      was registered in Denmark as an NGO network working on European level.
     </p>
