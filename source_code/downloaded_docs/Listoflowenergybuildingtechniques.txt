   <h1 class="firstHeading" id="firstHeading" lang="en">
    List of low-energy building techniques
   </h1>

    <div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en">
     <p>
      <a class="mw-redirect" href="/wiki/Low-energy_building" title="Low-energy building">
       Low-energy buildings
      </a>
      , which include
      <a href="/wiki/Zero-energy_building" title="Zero-energy building">
       zero-energy buildings
      </a>
      ,
      <a href="/wiki/Passive_house" title="Passive house">
       passive houses
      </a>
      and
      <a href="/wiki/Green_building" title="Green building">
       green buildings
      </a>
      , may use any of a large number of techniques to
      <a href="/wiki/Energy_conservation" title="Energy conservation">
       lower energy use
      </a>
      .
     </p>
     <p>
      The following are some of the techniques used to achieve low-energy buildings, which excludes
      <a href="/wiki/Microgeneration" title="Microgeneration">
       energy generation
      </a>
      (microgeneration).
     </p>
